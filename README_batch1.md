# 抖音自动回复软件 - 批次一交付

## 项目概述

本项目是抖音网页版自动回复功能的Windows桌面程序，批次一交付包含完整的PyQt5基础界面框架。

## 界面布局

严格按照《制作说明》文档要求实现了四区域界面布局：

1. **顶部控制栏**（高度50px）
   - [开始监控] [停止] [暂停] [学习模式] [设置] 按钮

2. **左侧设置面板**（宽度300px）
   - 基础设置：抖音私信URL、检测间隔、回复内容
   - 元素定位设置：消息列表区域、新消息提示元素、聊天输入框、发送按钮
   - 高级设置：智能重试、操作延迟、工作时段

3. **中央监控区域**（主区域）
   - 实时预览：浏览器预览占位区域
   - 操作指示器：红色圆圈、绿色高亮、蓝色框说明

4. **底部日志面板**（高度200px）
   - 操作日志显示区域，包含示例日志

## 默认值设置

- 抖音私信URL：`https://www.douyin.com/im`
- 检测间隔：30秒
- 回复内容："您好，我已收到消息，稍后回复您"
- 智能重试：启用
- 操作延迟：启用
- 工作时段：09:00 - 21:00

## 运行环境

- Python 3.7+
- PyQt5 5.15.10+

## 安装与运行

1. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

2. 运行程序：
   ```bash
   python main.py
   ```

## 注意事项

- 批次一仅实现界面框架，无功能实现
- 浏览器预览区域为占位符，后续批次将集成实际浏览器控件
- 所有按钮和设置项目前仅为界面元素，无实际功能
